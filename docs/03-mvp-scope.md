# MVP Scope

## P0 Capabilities

- Register, login, logout, email verification, password reset architecture.
- Create organization and manage membership.
- Roles and tenant isolation.
- Create and update projects.
- Add knowledge items: documents, notes, decisions, meeting summaries, links, updates, customer information, processes, and risks.
- Upload PDF, DOCX, TXT, and Markdown documents.
- Extract searchable text and create document chunks.
- Search with keyword, metadata, project, type, date, and permission filters.
- AI assistant that answers only from authorized sources.
- Activity timeline.
- Dashboard with sample insight modules.
- Mock integration catalog.

## Exclusions

- Production-grade integrations.
- Advanced analytics.
- Billing.
- Mobile apps.
- Enterprise identity.
- Autonomous decision-making.

## MVP Validation Criteria

- A user can create an organization, add project context, upload documents, and ask a grounded question.
- The answer includes cited sources and acknowledges missing evidence.
- A viewer cannot access restricted project or document context.
- The dashboard creates a credible weekly management view.
